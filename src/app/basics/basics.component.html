What is a HTML tag?

An HTML tag is a set of keywords wrapped in the < > brackets. This makes it possible for the browser to read and understand how to display elements. Back in the day when the Internet came around, these various tags were the only thing people had to create a website. And each tag served a different purpose.

These vary from image tags, to block tags, to heading tags and many many more tags.

This is a DIV tag, mostly used to group a collection of elements: {{this.example.html_2}}
Note that there are 2 div tags. The first one {{this.example.html_3}} tells the browser that that type of element is starting, and {{this.example.html_4}} to mark the element's end. The slash / tells the browser that it is the end div type element in this case.

A browser starts reading the document, sees there is a div tag. Then it considers everything after that to be the content of the div. Until it find the corresponding closing div. Most browsers are smart enough to close these for you, but try and remember the importance of it. For instance AngularJS 4 will break if it finds a mismatching set of tags.

These tags can be nested inside each other. Often you do need to do this to achieve a certain look.
