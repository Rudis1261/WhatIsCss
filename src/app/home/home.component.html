What does CSS stand for?

CSS is an acronym for "Cascading Style Sheets"

Websites

Most websites are made up of multiple technologies. Usually each one of these technologies have their own set of rules you will need to learn. Some technologies are required for the data (BackEnd) and some are required for the (FrontEnd) what you see and click.

To get started in Web Development, you will need to familiarize yourself with these:
  • HTML
  • CSS
  • JavaScript (Optional, but not really)

These are the core building block of a Web Page.

So what part does CSS play?

HTML documents defines a Web Page layout and general structure of the page. HTML documents consists of many HTML tags, either side by side or nested in each other.

CSS gives you the ability to style these elements, and have them look exactly the way you would like them to. It all comes together to create the website you see in your browser.
